# 🆘 Help Menu

This document lists quick help topics available within the application. Icons correspond to the sidebar entries or menu buttons.

- 🏗️ **Template Builder** – Tips on creating and editing workout templates.
- 🏋️ **Exercise Editor** – Manage exercises and muscle group assignments.
- 📆 **Calendar** – Plan workouts using day, week, month, and year views.
- 📝 **Activity Logs** – Review past sessions and filter by date.
- 📊 **Metrics** – View progress charts and analytics.
- 🖐️ **Muscle Browser** – Explore muscle groups and typical actions.
- ⚙️ **Settings** – Update profile details and application preferences.
- ❓ **Troubleshooting** – See [Logs and Troubleshooting](LOGS_AND_TROUBLESHOOTING.md) for common fixes.
