# Milestone 01: Metabolic Core

## Objective
Implement the metabolic domain core for VO2 max and lactate threshold workflows.

## Scope
- Protocol-aware VO2 max estimation contracts.
- Lactate sampling schema and threshold computation pipeline.
- Formula provenance and reproducibility metadata.
- Validation and error envelope standards for testing inputs.

## Deliverables
- Contract updates in shared schema modules.
- Backend service handlers with deterministic calculation paths.
- Frontend input flows for protocol selection and results review.
- Test coverage for protocol variants and boundary conditions.
